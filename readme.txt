
The dataset is shared in both csv and xlsx formats. You can also directly download the zip file to get them both.

1st row refers to randomly generated unique identifiers.

5 sentiment classes: Strongly Positive (SP), Strongly Negative (SN), Neutral (NU), Weakly Positive (WP), Weakly Negative (WN)

Size: 20,000 Bengali sentences (5,000 per sentiment class)



Please cite this paper for using the dataset:

M. S. H. Mukta, M. A. Islam, et al., "A Comprehensive Guideline for Bengali Sentiment Annotation", ACM Transactions on Asian and Low-Resource Language Information Processing (TALLIP), 2021.
